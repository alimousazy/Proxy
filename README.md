# Proxy
Proxy server which used to proxy request http request to another server.

Usage:
npm install -g babel 
npm install 
npm start

Arguments:

--url 
ex: 
npm start --- --url http://127.0.0.1:80/

--host
ex: 
npm start --- --host 127.0.0.1

--port
ex: 
npm start --- --port 8080

--log
ex: 
npm start --- --log logfile.txt
